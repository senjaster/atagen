# Наталья Николаевна Гасова
(1925–1997)

Дочь Николая Дмитриевича Бирюкова и [Татьяны Стоиловны Бойчевой](../B2/TSB.md), жена Бориса Петроовича Гасова, мать Татьяны Гасовой.
